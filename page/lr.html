猎人手游

简介 Summary

《猎人》手游是一款由《HUNTER×HUNTER（猎人×猎人）》（粉丝常用名：全职猎人）动画制作委员会正版IP授权、腾讯游戏发行、蓝图游戏研发、创想者株式会社辅助监修的横版动作手游，该作于2019年12月4日正式公开测试。

类型：即时2.5D格斗游戏，可收集装备，伙伴，升级技能，同职业亦拥有不同技能树。

服务器开发 Server Develop

签到模块开发 Sign Module

后台业务开发，在现有框架基础上，使用框架提供的API，完成功能开发，主要涉及信息存储和查询，前后台信息交互。

腾讯防沉迷系统接入 Tencent Anti-Addiction System

在充分了解现有框架后，单独配置一个功能服务进程与腾讯系统进行TCP通讯。注：后台架构有多个服务进程，包括客户验证，主进程，战斗房间进程，地图进程等，一般与外部信息的交换都会单独起一个进程服务，以防止阻塞主进程或导致整个服务崩溃。

玩家1V1开房模块开发 Match Module

设计开发内容主要包括： 1. 设计开房大厅玩家进入，创建房间，离开时通知规则； 2. 设计房间的数量，属性和接口； 3. 设计玩家在大厅内的状态变化；
